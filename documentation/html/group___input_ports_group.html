◆ ClockReceived() [1/2]

bool InputPort::ClockReceived ( float  clkDiv,
int8_t  bitShift 
)

Every clock received, recalculate interval for MIDI Clock sent.

Parameters
[in]floatclkDiv Clock Divider
[in]int8_tbitShift Clock shift
Returns
true when Clock processed

This functions perform two main tasks

  1. Calculate new interval for clock MIDI
    The interval is defined by two consecutive calls to this function, so the first call returns false.
    The calculated interval is filtered to avoid changes on rate due to small variations.
  2. Send the first clock message.
    When ClockDivider is 1 this is the only message sent
    See DigitalPort::ReadPort() and AnalogPort::ReadPort() for more details on processing when ClockDivider is bigger than 1

◆ ReadPort() [2/3]

bool AnalogPort::ReadPort ( long &  NewData)

Read analog Port make conversion and check analog clock.

Parameters
[in]NewDatawill store Read data
Returns
true when data read and converted has changed

This functions perform two main tasks

  1. Read the Analog port:
    1.1 Set the output for Bipolar input
    1.2 Read the analog input
    1.3 Filter the input values
    1.4 Convert data to MIDI as defined in the config data (see RangeConv::invConvert()
    1.5 Apply Amplification and Offset
  2. Send clock when ClockDivider is bigger than 1 (first clock is sent on InputPort::ClockReceived())
    It will check if the interval from last clock has passed and decrease the ClockCounter
    See InputPort::ClockReceived() for more details
