enum ICFun_e

Enumerator
INDEP 

Each Input acts independently.

SUM 

Sum CV and Fader Inputs.

MULTIPL 

Multiply CV anf Fader inputs. If Fader is set to Percent, it will act as percentage of CV.

NOTEMODE 

Note Mode: CV is the pitch, Fader is the velocity and Gate is the trigger.
